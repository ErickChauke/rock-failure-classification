# Rock Failure Mode Classification

A supervised machine learning project that classifies failure modes in rock specimens tested under laboratory compression conditions. Failure mode codes (2B, 3B, 4B, XA, YA) are assigned by trained technicians after each test. This project trains a model to predict those codes from measurements already recorded during the test.

## Dataset

Data comes from two laboratory test programmes conducted on four rock types: Footwall Pyroxenite, Gabbro Norite, Hangingwall Pyroxenite, and Reef.

| Test | Specimens (after cleaning) | Target |
|---|---|---|
| Triaxial compression (TCM) | 310 | Failure mode code |
| Uniaxial compression (UCM) | 137 | Failure code |
| Brazilian tensile (UTB) | excluded - no failure codes | - |
| Rock mass rating (RMR) | excluded - field mapping data | - |

Combined classification dataset: 439 specimens, 5 classes (2B, 3B, 4B, XA, YA).

Raw data files are not committed to the repository. Place the original XLS files in `data/raw/` before running the notebook.

## Results

| Model | Test Accuracy | F1 Weighted | F1 Macro |
|---|---|---|---|
| Logistic Regression | 0.545 | 0.613 | 0.377 |
| SVM | 0.682 | 0.717 | 0.456 |
| Random Forest | 0.795 | 0.761 | 0.375 |
| Random Forest (tuned) | 0.807 | 0.781 | 0.410 |

The tuned Random Forest achieves the highest overall accuracy and weighted F1. SVM achieves the highest macro F1, meaning it handles minority classes better when all classes are weighted equally.

Top features by SHAP importance: axial strain at failure, slenderness ratio, failure load, specimen height, test type.

## How to Run

1. Place the four original XLS files in `data/raw/`
2. Open `notebook.ipynb` in Jupyter
3. Run all cells from top to bottom - Section 1 installs any missing libraries automatically

## Limitations

- Classes 2B and 4B have too few specimens to be learned reliably (12 and 20 respectively)
- No raw stress-strain curve data is available - only summary statistics at 50% of peak load
- CNN and RNN approaches require curve data and are deferred to future work
